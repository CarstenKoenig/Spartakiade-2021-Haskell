auf der [Spartakiade 2021](https://spartakiade.org/)

## Organisatorisches

- Dauer **9:00** bis **16:00**
- Mittagspause **12:30** bis **13:30**

- Bitte keine Screenshots/Bilder der Teilnehmer auf Social Media teilen - danke!

- Einführung / Wiederholung
